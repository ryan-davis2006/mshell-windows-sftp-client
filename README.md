## What Mshell Is

Mshell is designed for Windows users who want one place to open SSH sessions and manage remote files through SFTP. Instead of splitting terminal work and file operations across separate tools, it keeps both available in the same application.

It is aimed at everyday remote administration, command-line maintenance, and moving data between local and remote systems. The emphasis is on reliable SSH access and SFTP navigation, without adding unrelated extras.
